# Grain of Life

Support: Linux, Windows

## Building for Linux

```bash
mkdir build
cd build
cmake ..
make
```
(Binary ends up at `grain-of-life/build/grain-of-life`)

## Building for Windows

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
(Binary ends up at `grain-of-life/build/Release/grain-of-life.exe`)
